## 📋 TASKS IN SEQUENCE

### TASK B1 — BUILD THE FASTAPI BACKEND
**Goal:** Serve the ML predictions and geographic boundaries to the frontend.

1. Navigate to the `backend/` folder.
2. Create a `main.py` file using FastAPI.
3. Add CORS middleware to allow the React frontend (usually running on `http://localhost:5173` or `3000`) to access the API.
4. Create the following endpoints:
   - `GET /api/health` -> Returns `{"status": "healthy"}`
   - `GET /api/predictions` -> Reads and returns the contents of `backend/predictions.json`.
   - `GET /api/geojson` -> Reads and returns `data/raw/kazakhstan_oblasts.geojson`.
   - `POST /api/simulate` -> **(Crucial for Hackathon Demo)** Accepts an Oblast name and a precipitation anomaly percentage (e.g., -50%). It must load `backend/model.pkl`, adjust the most recent precipitation lag by the given percentage, run live XGBoost inference, and return the new stress level and color.
5. Create a `requirements.txt` in the `backend/` folder (fastapi, uvicorn, xgboost, pandas, scikit-learn).

### TASK B2 — SCAFFOLD THE REACT FRONTEND
**Goal:** Set up a modern React app.

1. At the root of the project, run: `npx create-vite@latest frontend --template react` (or your preferred fast scaffolding tool).
2. CD into `frontend/` and run `npm install`.
3. Install necessary map and UI libraries: `npm install react-leaflet leaflet axios`. (You will use Leaflet to render the map).
4. **IMPORTANT:** Leaflet requires its CSS. Make sure to import `leaflet/dist/leaflet.css` in your `main.jsx` or `App.jsx`.

### TASK B3 — BUILD THE DASHBOARD UI
**Goal:** Build the interactive map and data table.

1. **The Map Component:** 
   - Use `react-leaflet` to display `kazakhstan_oblasts.geojson`.
   - The map must color each oblast based on the `color_30d` hex code provided in the `/api/predictions` endpoint.
   - Add tooltips/popups on hover showing the Oblast name, the 30/60/90 day forecast, and the confidence score.
2. **The Sidebar / Table Component:**
   - Display a list or table of the Oblasts that are currently in "Warning" or "Emergency" status so decision-makers can see them at a glance.
3. **The "What-If" Simulator (Live AI Inference):**
   - Add a slider allowing the user to select an Oblast and change the rainfall by a percentage (e.g., -100% to +100%).
   - When adjusted, send a request to `POST /api/simulate`.
   - Dynamically update that specific Oblast's color on the map based on the live AI response.
4. **Styling:**
   - Make it look like a premium, dark-mode command center. Think "mission critical satellite dashboard". Use a dark background (e.g., `#0f172a`) and bright, legible text.

### TASK B4 — FINAL INTEGRATION & RUN
1. Write a `start.sh` script in the root directory that starts BOTH the FastAPI server on port 8000 and the React frontend on its default port.
2. Provide a clear summary message stating that the app is ready for the hackathon demo.
